# Lento Logo Usage Rules v1

## 1) Aset Resmi

### Primary
- **Wordmark:** `lento_wordmarkA_full.svg`
- **Lettermark (ikon):** `lento_lettermarkA_full.svg`

### Varian Warna
| File | Gunakan untuk |
|------|---------------|
| `*_full.svg` | Default (latar netral/paper) |
| `*_black.svg` | Background terang polos |
| `*_white.svg` | Background gelap (dark mode) |

---

## 2) Kapan Pakai Wordmark vs Lettermark

| Lebar | Logo |
|-------|------|
| < 180px | Lettermark |
| ≥ 180px | Wordmark |
| ≥ 320px | Dengan tagline |

**Penempatan:**
- **Wordmark** → navbar, landing page, dokumen
- **Lettermark** → favicon, app icon, sidebar collapsed, mobile header

---

## 3) Clear Space

Unit = diameter lingkaran "o" di wordmark / radius ring di lettermark

- Minimal **1x** ruang kosong di semua sisi
- Jika dekat elemen ramai → naikkan ke **1.5x–2x**

---

## 4) Minimum Size (Digital)

| Logo | Minimum |
|------|---------|
| Wordmark | 180px lebar |
| Wordmark + tagline | 320px lebar |
| Lettermark | 24px |
| Favicon | 32×32px |

---

## 5) Background & Kontras

| Background | Gunakan |
|------------|---------|
| Terang | `*_full` atau `*_black` |
| Gelap | `*_white` |

Target kontras: **≥ 4.5:1** (WCAG AA)

---

## 6) Do / Don't

### ✅ Do
- Gunakan file SVG asli
- Pertahankan proporsi (scale uniform)
- Gunakan varian warna yang disediakan
- Jaga clear space minimal 1x

### ❌ Don't
- Jangan stretch
- Jangan ganti warna sembarang
- Jangan kasih outline, shadow tebal, glow, 3D effect
- Jangan taruh tagline kalau ukuran kecil

---

## 8) Lockups

| File | Deskripsi |
|------|-----------|
| `lento_lockup_horizontal_id_*.svg` | Logo + tagline horizontal (ID) |
| `lento_lockup_horizontal_en_*.svg` | Logo + tagline horizontal (EN) |

| `lento_lockup_stacked_en_*.svg` | Logo + tagline stacked (EN) |
