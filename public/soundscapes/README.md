# Soundscapes Audio Assets

This directory contains ambient sound files for the Soundscapes feature.

## Required Files

Please add the following MP3 files to this directory:

1. **rain.mp3** - Rain ambience (~300s loop)
2. **cafe.mp3** - Coffee shop ambience (~300s loop)
3. **white-noise.mp3** - White noise (~300s loop)
4. **forest.mp3** - Forest birds (~300s loop)

## Requirements

- **Format:** MP3
- **Bitrate:** 96-128kbps (balance quality vs size)
- **Size:** Target ~2-3MB per file
- **Loop:** Seamless loop (fade in/out or natural loop point)
- **License:** Royalty-free (CC0 or equivalent)
